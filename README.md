## ğŸ› ï¸ Getting Started

1. **Clone the repo:**
   ```bash
   git clone https://github.com/gaurav-nm/v2.git
   cd v2
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run locally:**
   ```bash
   npm start
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
